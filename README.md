# Sinfully Prompted — Event Universe Generator V2

## Setup (do this once)

Make sure you have Node.js installed. If not: https://nodejs.org

Open this folder in VS Code, then open the terminal (Ctrl+` or View → Terminal) and run:

```
npm install
```

## Run it

```
npm run dev
```

Opens at http://localhost:5173

## Deploy publicly (free)

```
npm run build
```

Then drag the `/dist` folder to: https://netlify.com/drop

You get a public URL instantly. No account needed.

## Files

- `src/App.jsx` — the entire generator (edit this to make changes)
- `src/main.jsx` — do not touch
- `vite.config.js` — do not touch
- `index.html` — do not touch
